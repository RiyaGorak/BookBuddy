# 📚 BookBuddy

BookBuddy is a modern book-sharing and rental platform that helps users discover, lend, borrow, and manage books within their local community.

The application is designed with a clean and user-friendly interface inspired by modern mobile apps.

---

## 🚀 Features

### 📖 Book Discovery

* Browse nearby books
* Search books by title, author, or category
* Explore popular books and readers

### 🤝 Book Borrowing

* Request books from other users
* Select rental duration
* Send personalized requests

### 💬 Chat System

* Connect with book owners
* Discuss pickup and return details
* Real-time conversation interface

### 📚 My Books

* Manage listed books
* Track borrowed books
* Monitor lending status

### 👤 User Profile

* View reading statistics
* Manage interests
* Access account settings

### ⭐ Community Features

* User ratings and reviews
* Reader profiles
* Book recommendations
